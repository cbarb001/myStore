# My Store - Web App


### An online store where the seller is able to:
* Post items to sell
* Edit items
* Delete items
* Categorize items

### and the buyer is able to:
* View items
* Add items to cart
* Remove items from cart
* Checkout

## Technologies:
 * HTML5
 * CSS3
 * Rails
 * Ruby
 * Bootstrap
 * PostgreSQL
 * Heroku
 * RSpec
